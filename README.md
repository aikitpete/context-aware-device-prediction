# pythonon-azure

A prototype project exploring how user preferences, location, weather, and travel context can be used to predict device settings.

## Overview
Python data-processing scripts with lightweight web files.

## What’s inside
Key files include:

- `GenerateData.py`
- `AnalyzeData.py`
- `ReduceFlightData.py`
- `ReduceWeatherData.py`
- `RequestResponse.py`
- `main.py`
- `Tutorial.py`
- `index.html`, `script.js`, `style.css`
- `Excel model.xlsx`

## Workflow
1. generate or collect source data,
2. reduce weather/travel data into usable features,
3. analyze that data,
4. test or expose prediction logic through lightweight scripts/web pages.

## Tech
- Python
- HTML/CSS/JavaScript
- spreadsheet-based modeling

## Why it matters
Relevant to IoT, personalization, and device automation conversations.
